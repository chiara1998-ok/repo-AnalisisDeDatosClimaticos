# repo-An-lisisDeDatosClim-ticos


Escenario A – Análisis de Datos Climáticos 
El equipo desarrollará un script que procese datos meteorológicos históricos de una ciudad. 
Objetivo 
Analizar un conjunto de datos climáticos para generar indicadores básicos de temperatura y 
precipitaciones. 
Finalidad del proyecto: 
● Importar un archivo CSV con registros de temperatura y precipitaciones. 
● Calcular indicadores como: 
○ temperatura promedio 
○ temperatura máxima y mínima 
○ promedio de precipitaciones 
● Generar al menos un gráfico que represente la evolución de la temperatura en el 
tiempo. 
● Guardar los resultados del análisis en la carpeta /resultados.

## Descripción del Proyecto

El presente proyecto tiene como objetivo analizar datos históricos de temperatura global mediante técnicas básicas de estadística descriptiva y visualización de datos.

Para el desarrollo se utilizó un dataset de temperaturas globales, aplicando herramientas de gestión colaborativa y control de versiones como Git, GitHub, Google Colab y Jira.

## Estructura del Repositorio

* `/datos`: contiene el dataset utilizado para el análisis.
* `/scripts`: contiene los scripts desarrollados en lenguaje R.
* `/resultados`: contiene gráficos y resultados obtenidos durante el procesamiento de datos.
* `README.md`: documentación general del proyecto.

## Análisis Realizado

Se calcularon las siguientes medidas estadísticas:

* Media
* Mediana
* Varianza
* Desviación estándar

Además, se generaron gráficos que permiten visualizar la distribución y comportamiento de las variaciones de temperatura a lo largo del tiempo.

## Herramientas Utilizadas

## Ejecución

1. Clonar el repositorio.
2. Acceder a la carpeta `/scripts`.
3. Ejecutar el script de análisis en R.
4. Consultar los resultados generados en la carpeta `/resultados`.

## Resultados

Los resultados obtenidos incluyen medidas descriptivas de tendencia central y dispersión, junto con representaciones gráficas que facilitan la interpretación de la evolución de la temperatura global.
